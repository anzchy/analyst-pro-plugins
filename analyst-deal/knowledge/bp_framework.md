# BP 深度分析框架 (BP Analysis Framework)

> deal-analyst Phase 2 使用。对 BP 进行结构化拆解。

---

## 分析框架

### 1. 商业模式拆解

- **价值主张**: 解决什么问题？为什么现有方案不够好？
- **客户验证**: 是否有付费客户？LOI/MOU ≠ 订单
- **收入模型**: 单位经济模型（单价 × 量 × 频次）
- **毛利结构**: 硬件型 vs 软件型？边际成本如何？
- **网络效应/锁定**: 客户切换成本？技术锁定？

### 2. 估值锚点

- **可比公司**: 同赛道已上市/已融资公司估值
- **可比交易**: 近期同领域融资案例
- **关键倍数**: PS / PE / EV/EBITDA（视阶段选择）
- **下行保护**: 如果最差情况，残值几何？

|---|---|---|---|---|

### 4. 风险红旗检查

### 5. 硬科技专项（如适用）

### 6. TRL 技术成熟度评估

> 参考 `knowledge/tech_checklist.md` §通用核查项·技术成熟度 获取赛道特定基准值和 RED FLAG 阈值。

| TRL 等级 | 定义 | 本项目状态 | 证据 |
|---------|------|-----------|------|
| TRL 1–3 | 基础研究/概念验证 | | |
| TRL 4–5 | 实验室验证/相关环境验证 | | |
| TRL 6 | 系统原型在相关环境中验证 | | |
| TRL 7–9 | 系统完成/量产就绪 | | |

- 当前 TRL 等级: [X]
- 达到下一等级的关键里程碑: [...]
- 预计时间和资金需求: [...]
- 2026 Series A 门槛参考: TRL 6+（多数深科技 VC 要求）

### 7. 供应链 & 量产可行性分析

> 参考 `knowledge/tech_checklist.md` §通用核查项·量产可行性 获取详细核查清单。

- **BOM 透明度**: 关键物料清单是否完整？是否通过独立工程审计？
- **供应商集中度**: 关键零部件是否依赖单一供应商？
- **FOAK（首台套）融资策略**: 股权/补贴/债务的混合比例
- **模块化扩产能力**: 能否以标准化单元部署，还是需要定制化设施？
- **目标毛利**: 量产阶段毛利率预期（深科技参考: 40%+）
- **制造合作伙伴**: 已有代工/制造合作还是需要自建产线？

### 8. IP 知识产权全景

> 参考 `knowledge/tech_checklist.md` §通用核查项·知识产权与合规 获取 FTO/出口管制核查项。

- **专利组合**: 已授权/已申请/计划申请数量
- **专利引用分析**: 被引用频次、核心专利影响力
- **FTO (Freedom-to-Operate)**: 是否存在阻断性专利？侵权风险评估
- **非专利壁垒**:
  - 数据飞轮（训练数据壁垒）
  - 制造秘诀（工艺 know-how）
  - 量化护城河（如"每 1000 小时运行，误差率下降 15%"）
- **专利战略 vs 保密策略**: 公开保护 vs 商业秘密，哪个更适合？

### 9. 团队构成风险评估

> 参考 `knowledge/red_flags.md` §团队红旗 获取完整的团队风险检查项。

| 角色 | 姓名 | 背景类型 | 产业经验(年) | 关键判断 |
|------|------|---------|-------------|---------|
| CEO | | 学术/产业/连续创业 | | |
| CTO | | 学术/产业/连续创业 | | |
| COO | | 学术/产业/连续创业 | | |

- **产业基因评估**: 团队是否有量产/工厂运营/供应链管理经验？
- **学术-产业平衡**: 纯学术团队需评估是否已补充产业化 COO
- **团队稳定性**: 核心成员在位时长，是否有近期离职
- **学术谱系**: 核心技术人员的学术脉络（导师、实验室、论文影响力）
- **耀途标准参考**: "不是第一梯队的技术团队直接 pass"

### 10. 国产替代定位分析

> 参考 `knowledge/red_flags.md` §合规红旗 获取出口管制相关检查项。

- **替代目标**: 替代哪家国际巨头的哪款产品？
- **替代进度**: 已验证/小批量导入/批量供货？
- **国产替代率**: 该品类当前国产化率是多少？
- **政策支持**: 是否列入"卡脖子"清单或重点攻关方向？
- **元禾璞华参考**: 80–90% 项目聚焦国产替代，10–20% 追求创新突破
- **出口管制影响**: 该技术/产品是否受 BIS Entity List 或中国出口管制影响？

### Stage-Aware Evaluation Lens（阶段化评估视角）

| 阶段 | 重点评估维度 | 次要维度 |
|------|------------|---------|
| 种子/天使 | 创始人领导力、技术创新性、市场洞察 | 短期财务不重要 |
| Pre-A / A 轮 | TRL 6+、团队产业基因、客户验证 | 供应链初步方案 |
| 量产爬坡期 | 供应链能力、单位经济模型、量产良率 | 规模化扩张路径 |
| Pre-IPO | 行业地位、持续增长能力、合规完备性 | 资本市场适配性 |
